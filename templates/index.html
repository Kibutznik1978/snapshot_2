✂️ How to Use the "Old Snapshot 2.0" Tool

  1. Go to: Schedule Bid Summary → Detailed Bid Summary
  2. Highlight everything starting from your name at the top of the page.
  3. Copy all the text — from your name and bid numbers down through the entire seniority bid table.
  4. The last line you copy should end with the last number in the last person's bid list.
  5. Paste all the copied text into the box below and click "Process Bids".
